# Usage
<!-- usage -->
```sh-session
$ npm install -g ftio-sdk
$ ftio COMMAND
running command...
$ ftio (-v|--version|version)
ftio-sdk/0.0.0 darwin-x64 node-v12.18.3
$ ftio --help [COMMAND]
USAGE
  $ ftio COMMAND
...
```
